# Vanto CRM

**WhatsApp AI CRM for MLM & APLGO Associates**

🌐 **Live:** https://chat.onlinecourseformlm.com  
📦 **Repository:** https://github.com/Vantovant/chat-friend-crm

---

## Overview

Vanto CRM is a production-grade, AI-powered WhatsApp CRM purpose-built for MLM professionals and APLGO associates. It centralizes lead management, WhatsApp conversations, pipeline tracking, group campaigns, and AI-driven automations into a single platform.

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React 18 · TypeScript 5 · Vite 5 · Tailwind CSS v3 · shadcn/ui |
| Backend | Lovable Cloud (Supabase) — PostgreSQL, Auth, Edge Functions, Realtime |
| Messaging | Twilio WhatsApp Business API |
| AI | Lovable AI Gateway (Google Gemini 2.5 Flash) + BYO OpenAI option |
| Extension | Chrome Manifest V3 (WhatsApp Web overlay) |
| External CRM | Zazi CRM (webhook sync) |
| State | TanStack React Query v5 |
| Charts | Recharts |
| Routing | React Router DOM v6 |

---

## Modules

| Module | Description |
|--------|-------------|
| **Dashboard** | KPI cards, lead temperature breakdown, message trends, activity feed |
| **Inbox** | Shared WhatsApp inbox with reply, status tracking, AI copilot |
| **Contacts** | Full CRUD, bulk actions, merge duplicates, CSV export |
| **CRM** | Drag-and-drop Kanban pipeline with APLGO lead types |
| **Group Campaigns** | Schedule WhatsApp group posts via Chrome Extension |
| **Automations** | Trigger → Action rules for CRM events |
| **AI Agent** | Conversational AI assistant with Knowledge Vault context |
| **Knowledge Vault** | Document management for AI-grounded responses (strict/assisted modes) |
| **Playbooks** | Message templates with usage analytics |
| **Workflows** | Multi-step automated sequences |
| **Integrations** | Twilio, Chrome Extension, Zazi CRM, OpenAI |
| **Settings** | Profile, team management, AI config, auto-reply, security |

## Quick Start (Local Development)

```bash
git clone https://github.com/Vantovant/chat-friend-crm.git
cd chat-friend-crm
npm install
npm run dev
```

Requires Node.js 18+ and npm. The app connects to Lovable Cloud backend automatically.

---

## Chrome Extension

1. Navigate to `public/chrome-extension/`
2. Open `chrome://extensions` → Enable **Developer Mode**
3. Click **Load unpacked** → select the folder
4. Open WhatsApp Web — the Vanto sidebar appears

---

## Environment Variables

| Variable | Purpose |
|----------|---------|
| `VITE_SUPABASE_URL` | Supabase API endpoint (auto-provided) |
| `VITE_SUPABASE_PUBLISHABLE_KEY` | Supabase anon key (auto-provided) |

Edge Function secrets (Twilio, Zazi, etc.) are configured in Lovable Cloud.
