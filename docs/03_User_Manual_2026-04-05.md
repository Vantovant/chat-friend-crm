## 1. Getting Started

### 1.1 Logging In
1. Go to `https://chat.onlinecourseformlm.com`
2. Enter your email and password
3. Click **Sign In**
4. You'll be taken to the Dashboard

### 1.2 Creating an Account
1. Click **Create account** on the login page
2. Enter your full name, email, and password (min 6 characters)
3. Click **Sign Up**
4. Your account is auto-confirmed — you can sign in immediately

### 1.3 Forgot Password
1. Click **Forgot password?** on the login page
2. Enter your email address
3. Check your inbox for the reset link
4. Follow the link to `/reset-password` and set a new password

## 2. Dashboard

The Dashboard is your home screen showing real-time analytics.

### KPI Cards
- **Total Contacts**: Number of active (non-deleted) contacts
- **Conversations**: Total WhatsApp conversation threads
- **Messages**: Total messages sent and received
- **Unread**: Messages awaiting your attention

### Charts
- **Messages per Day**: 7-day trend showing messaging activity
- **Leads by Type**: Pie chart showing your APLGO lead distribution:
  - Prospect → New leads
  - Registered_Nopurchase → Signed up, no purchase
  - Purchase_Nostatus → Bought products, no status
  - Purchase_Status → Active with GO-Status (VIP)
  - Expired → Lapsed membership

### Temperature Breakdown
- 🔴 **Hot**: Ready to buy/join — follow up immediately
- 🟡 **Warm**: Interested but needs nurturing
- 🔵 **Cold**: Low engagement — long-term prospect

## 3. Inbox (Shared WhatsApp Inbox)

The Inbox is where you read and reply to WhatsApp conversations in real-time.

### 3.1 Viewing Conversations
- Left panel shows all conversations sorted by most recent
- Each entry shows: contact name, last message preview, time, unread badge
- Click a conversation to open the message thread

### 3.2 Filtering Conversations
Use the filter tabs at the top:
- **All Accessible**: All conversations you can see (admins see all)
- **Mine**: Conversations assigned to you
- **Unassigned**: Conversations not yet assigned to any agent

### 3.3 Sending Messages (Reply Inbox)
1. Select a conversation
2. Type your message in the input box at the bottom
3. Press **Enter** or click the Send button
4. Your message is sent via WhatsApp Business API (Twilio)

### 3.4 Message Status Icons
| Icon | Meaning |
|------|---------|
| 🕐 Clock | Queued — waiting to send |
| ✓ Single check | Sent — delivered to Twilio |
| ✓✓ Double check | Delivered — reached the user's phone |
| ✓✓ Blue | Read — user opened the message |
| ⚠️ Red | Failed — click to retry |

### 3.5 Re-sending Failed Messages
- Failed messages show a red warning icon
- Click the retry button (🔄) to re-send

### 3.6 AI Copilot
- Click the **Brain** icon to open the AI Copilot sidebar
- The copilot suggests reply options based on conversation context and your Knowledge Vault
- Click a suggestion to insert it into the message box

### 3.7 Keyboard Shortcuts
| Key | Action |
|-----|--------|
| `Enter` | Send message |
| `Shift + Enter` | New line |

### 3.8 Contact Details
- Click the **Info** icon to view/edit the contact's details
- View: temperature, lead type, tags, notes, assignment, phone
- Reassign the conversation to another team member

## 4. Contacts

### 4.1 Adding a Contact
1. Click the **+ Add Contact** button
2. Fill in: Name (required), Phone (required), Email, Temperature, Lead Type, Interest

### 4.2 Searching & Filtering
- Use the search bar to find contacts by name, phone, or email
- Filter by temperature (hot/warm/cold), lead type, or interest level

### 4.3 Editing a Contact
- Click on a contact to open the detail drawer
- Edit any field and save

### 4.4 Bulk Actions
1. Select multiple contacts using checkboxes
2. Choose an action: **Delete**, **Add Tag**, or **Export CSV**

### 4.5 Merging Duplicates
1. Click **Merge Contacts** button
2. The system auto-detects duplicates by phone number
3. Select which record to keep as primary
4. Confirm the merge

## 6. Group Campaigns

### 6.1 Prerequisites
- Install the Vanto Chrome Extension
- Open WhatsApp Web in the same Chrome browser
- Ensure the extension shows **Connected** status

### 6.2 Capturing WhatsApp Groups
1. Open WhatsApp Web
2. The Chrome Extension automatically detects your groups
3. Use the extension's **Save Group** button to capture groups to the CRM

### 6.3 Scheduling a Single Post
1. Select a group from the dropdown
2. Type your message
3. Pick a date using the calendar popover
4. Set the time
5. Click **Schedule Post**

### 6.4 Bulk Campaign (Multi-Day)
1. Toggle **Bulk Campaign** mode on
2. Select a date range (from → to)
3. Check the time slots: Morning (08:00), Mid-day (13:00), Evening (18:00)
4. The system creates one post per day × selected time slots
5. Click **Schedule Bulk Campaign**

### 6.5 Monitoring Posts
- The scheduled posts table shows status for each post
- **Pending**: Waiting for scheduled time
- **Executing**: Chrome Extension is sending now
- **Sent**: Successfully delivered
- **Failed**: Shows failure reason — click **Retry** to try again

### 6.6 Troubleshooting
- If status shows "Extension Not Detected": ensure WhatsApp Web is open and the extension is active
- If posts fail at `[find_group]`: verify the group name matches exactly
- If posts fail at `[execution_timeout]`: check that WhatsApp Web is not minimized

## 7. Knowledge Vault

### 7.1 Adding Knowledge
1. Go to Knowledge Vault
2. Select a collection (e.g., "Product Prices")
3. Click **Upload File** (text or markdown) or **Paste Text**
4. Set the mode:
   - **Strict**: AI must quote verbatim from this content
   - **Assisted**: AI can paraphrase naturally
5. The file is automatically chunked and indexed for AI search

### 7.2 Collections
| Collection | Use For |
|------------|---------|
| General Knowledge | General info, FAQs |
| Business Opportunity | APLGO business model, joining process |
| Compensation | Commission structures, bonuses |
| Product Prices | Product pricing, packages |
| Orders & Deliveries | Shipping, order status |
| MLM Motivation | Motivational content for team |

### 7.3 How AI Uses Knowledge
- When a WhatsApp contact messages your number, the AI auto-reply searches the Knowledge Vault
- Strict collections (Products, Compensation, Orders) require exact quotes
- Assisted collections allow natural paraphrasing

## 8. AI Auto-Reply (WhatsApp)

When someone messages your WhatsApp Business number:

1. **First message (or after 4h silence)**: They receive the welcome menu:
   ```
   Hi 👋 Thanks for messaging Get Well Africa.
   Reply:
   1️⃣ Prices & Product info
   2️⃣ How to use / Benefits
   3️⃣ Speak to a person
   ```

2. **Reply "1"**: AI searches Product Prices + Business Opportunity collections and responds with pricing info
3. **Reply "2"**: AI searches Products + General Knowledge and responds with usage/benefits info
4. **Reply "3"**: Sends "Thank you. A team member will assist you shortly." — human handover
5. **Any other question**: AI detects intent (business/product/freeform) and searches relevant collections

### Rate Limiting
- Maximum 3 auto-replies per day per contact
- 10-minute cooldown between replies
- After limits: messages go to inbox for human response

## 11. Chrome Extension

### Installation
1. Download from the Integrations page or `public/chrome-extension` folder
2. Go to `chrome://extensions` → Enable Developer Mode
3. Click **Load unpacked** → select the extension folder
4. Pin the extension to your toolbar

### Usage
1. Open WhatsApp Web (`web.whatsapp.com`)
2. The Vanto sidebar appears on the right
3. Click on any chat → the extension detects the contact
4. Use the sidebar to:
   - Save contacts to CRM
   - Add notes
   - View temperature and pipeline stage
   - Get AI reply suggestions
   - Capture WhatsApp groups for campaigns
